# CRUD System ⚡ — Route Academy

This project is part of my Frontend Development journey at **Route Academy**.

This is my **first real JavaScript project** — a fully functional CRUD System that adds, updates, deletes, and searches products with data persistence using Local Storage.

---

## 📌 Project Goal

Build a fully functional CRUD (Create, Read, Update, Delete) system using Vanilla JavaScript — moving beyond static HTML/CSS pages into interactive, data-driven web applications.

---

## 🧱 What Makes This Different

All previous projects were static pages. This is the first project where the page actually **responds, remembers, and reacts** to user input using JavaScript and the Web Storage API.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome Icons
- **JavaScript (Vanilla JS)**
- **Web Storage API (Local Storage)**

---

## 📚 What I Practiced

- DOM Manipulation
- Event Handling (oninput, onclick, onchange)
- Form Validation with Regex
- CRUD Operations in JavaScript
- Local Storage (save, retrieve, update, delete)
- Dynamic rendering of product cards
- Real-time search functionality

---

## ✨ Features

| Feature | Description |
|---|---|
| ➕ Add Product | Add a new product with name, price, category, description, and image |
| ✏️ Update Product | Edit any existing product and save changes |
| 🗑️ Delete Product | Remove any product from the list |
| 🔍 Search | Real-time search by product name |
| 💾 Local Storage | Data persists after page refresh |
| ✅ Validation | Live form validation with error messages |
| 🖼️ Image Upload | Upload an image per product |
| 📋 Category | Select category from datalist (tv, mobile, screens, electronic) |
