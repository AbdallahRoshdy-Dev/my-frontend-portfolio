# Clarity 💻 — Route Academy

This project is part of my Frontend Development journey at **Route Academy**.

The goal was to build a full Digital Solutions Platform with advanced Bootstrap 5 components — the most complex project so far.

---

## 📌 Project Goal

Build a fully structured digital agency website with multiple advanced sections including a Mega Dropdown Navbar, Hero, Services, Technology Stack Tabs, Testimonials Carousel, FAQ Accordion, Contact Form, and Footer.

---

## 🧱 Layout Approach

- ✅ **Bootstrap 5** — Navbar, Tabs, Carousel, Accordion, Grid, Forms
- ✅ **Flexbox** — used alongside Bootstrap for custom components
- ✅ **Scroll Spy** — for active navbar link tracking
- ✅ **Media Queries** — for full responsiveness

Not used:
- ❌ CSS Grid
- ❌ Float / Inline-block

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome Icons
- JavaScript (Bootstrap Bundle)

---

## 📚 What I Practiced

- Mega dropdown navigation menu with categorized services
- Bootstrap Pills/Tabs for dynamic content switching
- Accordion for FAQ sections
- Testimonials Carousel with 5 client reviews
- Advanced dual-column contact form layout
- Consistent design system with custom utility classes
- Media queries for full responsiveness

---

## 🗂️ Sections Built

| Section | Description |
|---|---|
| Navbar | Fixed top, mega dropdown Services menu, Sign In & Get Started buttons |
| Hero | Headline, CTA buttons, 3 metrics (500+ Projects, 98% Satisfaction, 24/7 Support) |
| Services | 6 service cards with icons and hover effects |
| Technology Stack | Tabbed section — Frontend, Backend, Mobile, Cloud & DevOps, Database |
| Testimonials | Bootstrap Carousel with 5 verified client reviews and ratings |
| FAQ | Accordion with 6 collapsible questions |
| Contact | Dual-column — consultation form + Why Choose Us sidebar |
| Footer | Logo, description, and social links |
