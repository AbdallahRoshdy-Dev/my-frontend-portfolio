# ContactHub 📒 — Route Academy

This project is part of my Frontend Development journey at **Route Academy**.

A fully functional Smart Contact Manager — add, organize, and manage your contacts with favorites, emergency contacts, real-time search, and data persistence.

---

## 📌 Project Goal

Build a real-world contact management app using Vanilla JavaScript with full CRUD operations, toggle states, live search, stats dashboard, and Local Storage persistence.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (Grid, Forms)
- Font Awesome Icons
- **JavaScript (Vanilla JS)**
- **Web Storage API (Local Storage)**

---

## 📚 What I Practiced

- Advanced DOM Manipulation
- Dynamic rendering of contact cards
- Real-time search & filtering
- Toggle states (Favorite / Emergency)
- Live stats counter updates
- Form validation with Regex
- Local Storage CRUD operations
- Sticky sidebar layout
- Modal open/close interactions

---

## ✨ Features

| Feature | Description |
|---|---|
| 📊 Stats Dashboard | Live counters for Total, Favorites, and Emergency contacts |
| ➕ Add Contact | Full form with photo upload, name, phone, email, address, group, notes |
| ✏️ Edit Contact | Update any contact's information |
| 🗑️ Delete Contact | Remove contacts from the list |
| ⭐ Favorite Toggle | Mark/unmark contacts as favorites |
| 🚨 Emergency Toggle | Mark/unmark contacts as emergency |
| 🔍 Real-time Search | Search by name, phone, or email instantly |
| 📋 Sidebar | Quick access panel for Favorites & Emergency contacts |
| 📞 Call Button | Direct call link per contact |
| 📧 Email Button | Direct email link per contact |
| 🖼️ Photo Upload | Upload a custom photo per contact |
| 🏷️ Groups | Categorize contacts (Family, Friends, Work, School, Other) |
| 💾 Local Storage | Data persists after page refresh |
| 🈳 Empty State | Clean UI shown when no contacts exist |
| ✅ Validation | Live form validation with error messages |
