# GameArena 🎮 — Route Academy

This project is part of my Frontend Development journey at **Route Academy**.

The goal was to build a full gaming website with multiple sections, using HTML, CSS, and **Bootstrap 5** for the first time.

---

## 📌 Project Goal

Build a fully structured gaming website with multiple sections including a Navbar, Hero, Games Carousel, What We Do, Our Team, Contact Form, and Footer — practicing real-world layout patterns with Bootstrap 5.

---

## 🧱 Layout Approach

This project was built using modern CSS layout tools:

- ✅ **Bootstrap 5** — used for Grid, Navbar, Carousel, Forms, and responsive layout
- ✅ **Flexbox** — used alongside Bootstrap for custom components
- ✅ **Scroll Spy** — for active navbar link tracking

The following were not used:
- ❌ CSS Grid
- ❌ Float / Inline-block

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome Icons
- JavaScript (Bootstrap Bundle)

---

## 📚 What I Practiced

- Using Bootstrap 5 components for the first time (Navbar, Carousel, Grid, Forms)
- Building a multi-section page with clean structure
- Responsive layouts using Bootstrap Grid system
- Custom CSS styling on top of Bootstrap
- Building interactive UI components (Carousel, Scroll Spy)
- Contact form with dropdown, checkbox, and textarea
- Fixed navbar with responsive toggler

---

## 🗂️ Sections Built

| Section | Description |
|---|---|
| Navbar | Fixed top, responsive toggler, Sign In button |
| Hero | Headline, CTA buttons, Active Players badge |
| Our Games | Bootstrap Carousel with 9 games across 3 slides |
| What We Do | 8 service cards (PC, Mobile, AR/VR, PS4, 3D) |
| Our Team | 8 pro player cards with roles & achievements |
| Contact | Full form with dropdown, checkbox & social links |
| Footer | 4-column layout with social icons & policy links |
