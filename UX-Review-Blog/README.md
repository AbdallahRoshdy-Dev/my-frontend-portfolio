# The UX Review Blog – Route Academy

This project is part of my Frontend Development journey at **Route Academy**.
The goal was to build a multi-section blog layout using HTML and CSS with modern layout techniques.

---

## 📌 Project Goal

Build a fully structured blog page with multiple sections including a **Navbar**, **Hero**, **Articles**, **Authors**, **Community**, and **Footer** — practicing real-world layout patterns used in modern web development.

---

## 🧱 Layout Approach

This project was built using modern CSS layout tools:

- ✅ **Flexbox** — used for navbar, cards, article items, footer columns, and most wrappers

The following were **not** used:

- ❌ CSS Grid
- ❌ Float / Inline-block

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 📚 What I Practiced

Through this assignment, I practiced:

- Building a multi-section page (Navbar, Hero, Articles, Authors, Community, Footer)
- Flexbox for structured and responsive layouts
- Reusable utility classes (e.g. `box-shadow`, `white-color`, `brdr-btm`)
- Organizing CSS with consistent naming conventions
- Building card components and testimonial sections
- Newsletter input + subscribe button in the footer
- Working with icons and image wrappers inside components
