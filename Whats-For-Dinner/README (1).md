# What's For Dinner 🍽️ — Route Academy

This project is part of my Frontend Development journey at **Route Academy**.

A recipe app that fetches a random meal every time you click "Try Another Recipe" — because deciding what to eat shouldn't be hard!

---

## 📌 Project Goal

Build an API-powered recipe app using Vanilla JavaScript that dynamically renders meal data including ingredients, instructions, nutrition facts, and chef's tips.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (Tabs, Navbar, Grid)
- Font Awesome Icons
- **JavaScript (Vanilla JS)**
- **MealDB API**

---

## 📚 What I Practiced

- Fetching data from a public API (MealDB)
- Dynamic DOM rendering from API response
- Bootstrap Pills/Tabs with dynamic content
- Random item selection with JavaScript
- Conditional UI (showing/hiding alerts based on cook time)
- Clean component-based HTML structure

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎲 Random Recipe | Click "Try Another Recipe" to load a new random meal |
| 📋 Ingredients Tab | Numbered list of all ingredients with quantities |
| 📖 Instructions Tab | Step-by-step cooking instructions |
| 🥗 Nutrition Tab | Calories, Protein, Carbs, Fat, Fiber, Sodium breakdown |
| 💡 Chef's Tips Tab | Pro tips for best results |
| ⚠️ Time Alert | Warning shown automatically if cook time exceeds 45 minutes |
| 🔖 Bookmark | Save recipe button |
| 🔗 Share | Share recipe button |
| ⭐ Rating | Recipe rating and review count |
| 👤 Navbar | Responsive navbar with user avatar and action icons |

---

## 🗂️ Sections Built

| Section | Description |
|---|---|

| Recipe Details | Title, tags, tabs, and action buttons |
